Son of Blue WordPress Theme

written by Web Considerations
- October 31, 2008

The Son of Blue WordPress Theme is a new WordPress theme with a easy to read layout and colorful design including a custom menu and custom header logo.

If you need assistance with customizing the "Son of Blue" logo for your site name, contact me and I will be glad to help.

It is called Son of Blue because it was inspired by my original WordPress Theme - Blue Fade. Son of Blue, however, was designed not only for clean and crisp reading but more space has been added for your navigational links. An extra column has also been added in addition to the custom top menu bar so you will have plenty of space for your widgets, page and category links, calendars, advertisements and more.

If you have any questions or find any issues with this theme, please contact me, I will be more than happy to answer your questions and/or fix any problems that may exist.

Features

  • Clean and easy to navigate, three columns, Photo-friendly, Widget ready
  • Passes both XHTML/CSS validation
  • Gravatars support
  • Access to admin menu from any page
  • PNG file included so you can change the default “Son of Blue” text in the logo to match your blog name

Son of Blue Logo Text

The Son of Blue theme includes a “logor.png” file located in the images directory that you can edit to include your own blog name with the custom styling that is used on the “Son of Blue” logo. I used Fireworks to create the file but any graphics editor that can read png files should work.

To edit the file, open in your graphics editor.

  1. The file is wp-content\themes\son-of-blue\images\logo.png.
  2. Change the “Son of Blue” text to the name you want i.e. your blog name.
  3. Save the file and upload it to your server.
  4. You may also need to change the sytling for your site name and description text to center it under the logo. To change the text for the site name and description (under the logo), open the style.css. Look for "blogtitle" and modify the left: property - it is currently set at 45px;. Change it to a numeric value less or greater than 45px to center under the logo.
