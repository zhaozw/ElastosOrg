Join Elastos Open Source Project

在這裏，妳將可以：
Elastos.org is all love Elastos development, is willing with other partners in Elastos open source project developers hard, participating and contributing code, exchange experience, Elastos online downloads and documentation support, and the open source community to participate in community activities.

  • 了解Elastos.org現在正在進行的各個開源項目，並獲得相關代碼和資料。
  • 參與或者發起壹個開源項目，爲項目貢獻代碼。
  • 進入Review系統，查看所有項目所有人提交的代碼，Review系統將對所有人的代碼進行統計，計算您對項目的貢獻。
  • 進入Documents wiki文檔系統，獲取教程以及幫助信息，並且可以在wiki中創建新的條目，或者編輯、修改已有的wiki條目。
  • 管理您自己的個人帳戶，並通過個人的主頁了解您參與的群組或項目的最新動態，並進入您參與的群組、博客、論壇，訪問朋友的博客。

如何加入Elastos.org？

First,you shall get an elastos.org OpenID on Developer WebsiteElastos.org開源項目目前僅對受邀請的開發者開放，開發者可通過下面的步驟加入Elastos.org：
  • 訪問Elastos.org開源社區網站，了解Elastos.org開源項目的宗旨，了解Elastos.org開源活動的內容及有關介紹。
  • 向項目負責人或者管理員發出郵件，提出加入Elastos.org開源社區的申請。項目負責人會與您聯系，交流開發經驗和個人興趣。通過審核後將向您發出邀請郵件。
  • 根據接收到的邀請郵件中的鏈接，您可以在Elastos.org開源網站首頁註冊並獲得您的唯壹的ElastosID。
  • 根據xxxx中介紹的方法，獲得與您的本地工作機器邦定的公共密鑰，並在遠程的服務器上註冊密鑰。
  • Elastos.org開源社區包括項目管理系統、代碼review系統、文檔系統以及SNS社區服務如個人信息、群組、朋友、博客、論壇等多個需要註冊帳號的子門護。
  • Elastos.org開源社區建議您使用妳獲得ElastosID時使用的同壹帳戶信息以上子系統中註冊，並進壹步做個性化設置。

Elastos.org博客內容如何同步到新浪（或其他）微博？

在個人設置頁的同步邦定欄設置同步類型，選中新浪（或者其他）微博的復選框，當妳分享博客內容或者發表評論時，均可以同步到新浪（或者其他）微博。

如何管理您的個人帳戶，並參與社區活動？

    Activity & Blogs。
  • 動態列表
    • Sites
  • 您在Elastos.org社區參加的站點
    • Groups。
  • 創建、加入、成員、動態列表、離開
    • Friends & Messages
  • 邀請朋友、關註朋友、朋友列表、收發郵件
  • 在註冊 Elastos.org 時， Elastos.org 會先為妳推薦已經註冊的活躍的開發者，妳只用點擊關註，就可以第壹時間了解這些開發者在Elastos.org的最新動態；
  • 可通過MSN，Email，人人等方式，邀請妳的好友加入 Elastos.org ，收到邀請的好友在註冊 Elastos.org 後就會自動關註妳！讓妳迅速建立自己的虛擬社區網酪！

  • 在Forums中創建Topic、回復Topic、收藏喜愛的Topic。
  • 根據自己的喜好，將自己關心的Topic進行收藏，便於時常訪問。

  • 個人帳戶設置、名字、頭像。
  • 個人註冊郵箱、密碼。
